# Templates

本目录存放 PPT 模板文件（`.potx` / `.pptx`）。

## 当前状态

尚无模板文件。

## 模板使用策略

- 当用户提供企业模板时，将其放入此目录并使用 `template-placeholder` backend。
- 模板应包含定义好的 master slide、placeholder、主题色和字体。
- 不要在此目录存放生成后的 PPTX 文件，模板仅指输入模板。

## 与 backend 的关系

模板使用遵循 `references/backend-selection.md` 中的 `template-placeholder` 策略。
