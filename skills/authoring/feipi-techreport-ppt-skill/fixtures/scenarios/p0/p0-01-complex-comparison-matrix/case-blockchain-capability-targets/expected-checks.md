# Expected Checks: P0-01

## Must

- [ ] 页面只有 1 页。
- [ ] 使用 16:9 横版。
- [ ] 表格区域占页面 60% 以上。
- [ ] 至少包含 5 个对比对象。
- [ ] 至少包含 8 个维度。
- [ ] 表格为原生可编辑结构，不是截图。
- [ ] Whale 目标列有明显视觉高亮。
- [ ] 右侧包含"数据口径 / 潜在热点 / 页面结论"三个信息块。
- [ ] 没有明显文字溢出、重叠、被裁剪。
- [ ] 底部脚注或右侧口径能说明数据不确定性。

## Should

- [ ] 表格横向和纵向边界清晰。
- [ ] 关键数字如 10K+、100K+、500ms 应有视觉强调。
- [ ] 右侧说明不超过 3 个卡片。
- [ ] 数据口径使用短句，不写长段落。

## Must Not

- [ ] 不得把整页渲染成图片。
- [ ] 不得把表格做成不可编辑图片。
- [ ] 不得让右侧说明区抢占表格主体位置。
- [ ] 不得将每个单元格写成完整段落。
