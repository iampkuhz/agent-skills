# Page Contract: P0-01 复杂多对象对比矩阵页

## Layout

- 16:9 单页。
- 标题区高度不超过 12%。
- 主表格占页面 60%–75%。
- 右侧说明区占页面 20%–28%。
- 底部可保留 1 行脚注。

## Required Regions

1. Title region
2. Comparison matrix region
3. Side note region
4. Source / assumption note region

## Table Requirements

- 至少 5 个对比对象。
- 至少 8 个维度。
- 表格必须可编辑。
- 表头、行头必须清晰。
- 自建链 目标列必须高亮。
- 单元格文本必须短语化，不允许长段落。
- 单元格不得明显溢出边框。

## Side Note Requirements

右侧必须包含：

1. 数据口径 / 批注
2. 潜在热点
3. 页面结论

## Style

- 技术汇报风格。
- 蓝色为主色，自建链 目标列使用高亮底色。
- 允许少量强调色，但不能破坏表格可读性。
