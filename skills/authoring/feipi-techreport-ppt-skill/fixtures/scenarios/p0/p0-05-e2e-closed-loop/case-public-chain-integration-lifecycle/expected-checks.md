# Expected Checks: P0-05

## Must

- [ ] 页面只有 1 页。
- [ ] 左侧有系统链路图。
- [ ] 系统链路至少包含业务系统、适配层、访问层、公链网络、链上资产/合约。
- [ ] 右侧包含数据可信、密钥安全、资产/应用操作安全。
- [ ] 包含高 TPS 链 3 个重点投入项。
- [ ] 底部包含 5 步接链生命周期。
- [ ] 最底部包含接链完成标准。
- [ ] 页面表达的是生产闭环，不是单纯 RPC 接入流程。
- [ ] 没有明显文字溢出、组件重叠。

## Should

- [ ] 每个技术重点都有检查点和风险。
- [ ] 技术事项卡片有图标或醒目标识。
- [ ] 生命周期步骤有编号。
- [ ] 完成标准视觉上最醒目。

## Must Not

- [ ] 不得只画 RPC 到公链的简单箭头。
- [ ] 不得省略安全或数据可信。
- [ ] 不得把完成标准弱化成普通脚注。
