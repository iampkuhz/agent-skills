# Expected Checks: P0-04

## Must

- [ ] 页面只有 1 页。
- [ ] 包含 5 个横向阶段。
- [ ] 每阶段包含时间窗口。
- [ ] 每阶段包含核心 KPI / TPS 指标。
- [ ] 每阶段包含性能升级与金融增强。
- [ ] 阶段之间视觉上有演进关系。
- [ ] 关键 TPS 数字明显大于普通正文。
- [ ] 没有明显文字溢出。

## Should

- [ ] 顶部有一句结论。
- [ ] 阶段卡片等宽、对齐。
- [ ] 每阶段信息结构一致。
- [ ] 场景与生态支持可作为底部轻量层展示。

## Must Not

- [ ] 不得做成普通大表格。
- [ ] 不得让某一阶段显著高于/宽于其他阶段，除非明确强调。
- [ ] 不得省略阶段目标或时间窗口。
