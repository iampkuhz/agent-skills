# Source: 高性能 EVM 链部署架构与改造点图

生成 1 页 16:9 技术汇报 PPT。

主题：高性能 EVM 链部署架构与改造点图。

页面目标：说明一个高性能 EVM 公链的目标架构，包括验证者网络、共识层、执行层、OS/硬件/ECS 资源、RPC/索引/监控等链外节点，以及右侧的改造类型分类。

主图需要表达：

- 多个 Validator 节点，Validator 1 展开显示内部结构。
- Validator 内部分层：
  - 共识层：共识引擎、排序/出块、投票/QC、结果确认。
  - 执行层：交易池、组块器、EVM、并行执行、Gas、状态库。
  - OS / 硬件 / ECS：CPU、内存、存储、网络、内核。
- 改造点：
  - B1 共识算法优化
  - B2 网络传播优化
  - C1 客户端语言
  - C2 交易池/组块
  - C3 EVM执行
  - C4 存储优化
  - C5 费用优化
  - C6 交易受理改善
  - D1 RPC受理与处理
  - D2 硬件与OS

右侧需要是"改造类型（一级分类 → 二级分类）"，按 A/B/C/D 分组说明。

注意：

- 架构图重点是分层与组件关系，不要画成纯流程图。
- 组件数量较多，要详略得当。
- 改造点标签要和右侧分类对应。
