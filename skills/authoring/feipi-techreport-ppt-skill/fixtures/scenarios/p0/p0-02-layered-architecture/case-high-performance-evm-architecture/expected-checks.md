# Expected Checks: P0-02

## Must

- [ ] 页面只有 1 页。
- [ ] 主图是分层架构图，不是线性流程图。
- [ ] 至少包含共识层、执行层、OS/硬件/ECS 三层。
- [ ] Validator 1 展开，其他 Validator 可以简化。
- [ ] 包含 RPC 节点、索引节点、监控/日志。
- [ ] 图中改造点标签能对应右侧 A/B/C/D 分类。
- [ ] 没有明显文字溢出、组件重叠、箭头遮挡文字。

## Should

- [ ] 主图区和右侧说明区比例清晰。
- [ ] 每层组件不超过 8 个。
- [ ] 改造点标签视觉上比普通组件更突出。
- [ ] 右侧分类卡片层级清楚。

## Must Not

- [ ] 不得把所有组件平铺成无层级网格。
- [ ] 不得使用过多交叉箭头。
- [ ] 不得把右侧分类写成大段文字。
