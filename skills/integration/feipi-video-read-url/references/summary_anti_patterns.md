# 摘要反模式清单

## 禁止的套话/废话模板

以下表达不应出现在摘要结果中：

- 并承接前后文
- 围绕同一主题反复展开
- 文本信息相对连续，已按语义段落做合并
- 补充要点：围绕……展开延伸
- 网络资料显示
- 相关媒体报道
- 可能会产生一定影响
- 仅供参考
- 可能不准确，仅供参考
- 信息不足，无法全面覆盖
- 以上内容仅供参考
- 如有需要可以进一步了解

## 禁止的表达模式（证据相关）

- 把外部背景伪装成视频内观点
- 把模型判断伪装成视频原话
- 明明材料不足却输出确定性事实
- 用"可能不准确，仅供参考"作为结尾套话

## 禁止的时间戳模式

- 在无时间戳文本中编造 [MM:SS] 或 [HH:MM:SS]
- 使用 T+00:00:00 格式
- 使用字幕行号代替时间锚点
